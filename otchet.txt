Отчёт номер 2, о коммутаторе

Коммутатор был настроен с помощью пакета openvswitch, интерфейс ens33 был настроен как транковый для подсетей 100,200,999.
Интерфейс ens34 был настроен как тэгированный для сети 100, интерфейс ens35 был настроен как тэгированный для сети 200.
Сам коммутатор находится в сети 999.


Отчёт номер 3, о туннеле

Был сконфигурирован GRE-туннель между двумя роутерами HQ-RTR и BR-RTR, имеющий адрес 10.10.10.9 на первом роутере, и 10.10.10.10 на втором роутере. 
GRE был выбран в качестве туннеля из-за его универсальности и возможности передавать широковещательный трафик, что позволяет использовать через такой туннель протоколы маршрутизации.


Отчёт номер 4, о динамической маршрутизации

Для динамической маршрутизации был выбран OSPF. Его преимущества - автоматическая адаптация к изменениям в сети, быстрая сходимость, масштабируемость и поддержка иерархической структуры сети.


Отчёт номер 5, о DHCP

Был настроен DHCP на HQ-RTR и по MAC-адресу HQ-CLI были выданы:
Статический IP-адрес - 192.168.1.66
Маска подсети - 255.255.255.240 (/28)
Шлюз - 192.168.1.65
Днс-сервер - 192.168.1.2
Днс-суффикс - au-team.irpo


Отчёт номер 6, о файловом хранилище

Было настроено файловое хранилище и собран RAID-массив уровня 5 из трёх дисков по 1 гб с файловой системой ext4 на HQ-SRV. Так же был настроен сетевой файловый сервер на HQ-SRV и HQ-CLI.
У HQ-SRV сетевое хранилище располагается по пути /raid5/nfs, у HQ-CLI сетевое хранилище располагается по пути /mnt/nfs.


Отчёт номер 7, о moodle

Был настроен сервис moodle на HQ-SRV. Используется веб-сервер apache, в качестве бд используется mariadb, создана бд moodledb, задан пароль P@ssw0rd для пользователя admin в системе обучения.


Отчёт номер 8, о браузере

Был установлен Яндекс Браузер для организаций на HQ-CLI.


Отчёт номер 9, о Zabbix

В качестве системы мониторинга был выбран Zabbix за его простоту установки, гибкость настройки и бесплатное предоставление. Zabbix работает на порту 8080, так как он был развёрнут в контейнере Docker. Работает Zabbix по адресу http://mon.au-team.irpo:8080
